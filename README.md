# lipstick
搜集各种口红品牌，系列，色号

## 如何提交

提交到<code>lipsticks</code>目录下，像这样<code>品牌名/系列/名称.MD</code>，若不存在，就创建。
<code>名称.MD</code>参考格式如下

例如
>
>  | 名称 | 生产日期 | 色号 | rgb| 图片链接 |
>  | ---- | ------- | ----- | ----- | ----- |
>  | xxxxx | xxxxxxx | xxxxx | xxxxx | xxxxx |

<strong>注意</strong>：rgb不一定要准确。也可以在下面写一段评测哦^-^
